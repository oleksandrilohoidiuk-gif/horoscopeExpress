# AI Horoscope Generator
Веб-додаток для динамічної генерації персоналізованих гороскопів за допомогою штучного інтелекту, з можливістю збереження в базу даних, фільтрації та оцінювання. 

# Основний функціонал
AI Генерація: Створення унікальних гороскопів за допомогою інтеграції з Groq API.

Збереження: Уникнення дублікатів у базі даних — якщо гороскоп для певного знаку та місяця вже існує, він оновлюється, а не створюється наново.

Фільтрація: Зручний пошук гороскопів у базі за місяцем та знаком зодіаку.

Система реакцій: Можливість ставити реакції (🔥, 😐, 🗑️) згенерованим текстам без перезавантаження сторінки.
# Встановлення та запуск сервера

## 1. Встановлення моделей
`npm install`

## 2. Налаштування середовища
API_KEY=ваш_ключ_від_groq_api
DB_URL=ваша_база_даних

## 3. Запуск сервера
`npm start`

### Сервер запускається на http://localhost:3000
